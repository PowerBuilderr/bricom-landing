# BriCom — Landing Page

> **Портфолио-проект.** BriCom — вымышленный бренд строительной компании, созданный для демонстрации навыков вёрстки и UI/UX-дизайна.

## 🔗 Demo

👉 [Посмотреть сайт](https://powerbuilderr.github.io/bricom-landing/)

---

## О проекте

Одностраничный лендинг для строительной компании премиум-сегмента. Проект создан как портфолио-кейс: от концепции до готовой вёрстки.

**Что реализовано:**
- Hero с параллакс-эффектом фона и морф анимацией
- Секция «Цифры доверия» (анимированные счётчики)
- Блоки «Миссия / Подход», «Услуги», «Проекты»
- Раздел «Технологии и материалы»
- Секция отзывов
- Контактная форма
- Секция FAQ с аккордеоном
- Адаптивный Footer

## Технологии

| | |
|---|---|
| HTML5 | Семантическая разметка |
| CSS3 | Flexbox, Grid, CSS-переменные, анимации |
| Vanilla JS | Без фреймворков и сборщиков |
| Google Fonts | Cormorant Garamond + Jost |

---

## Особенности

- **Без зависимостей** — чистый HTML/CSS/JS, никакого npm
- **Адаптивная вёрстка** — корректно отображается на мобильных, планшетах и десктопе
- **Параллакс и анимации** — реализованы через Intersection Observer и CSS transitions
- **Тема** — минималистичный премиум-стиль
